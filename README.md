# Shadow Escape: Reborn

A production-grade 2.5D Side-Scrolling Action Platformer built entirely using raw, vanilla HTML5 Canvas technologies, structured natively without external engines or compilation configurations.

## Architecture

- **Engine Core:** central loop system utilizing fixed-time step integration loops processing state ticks independently from visual context renders.
- **Physics Engine:** continuous spatial matrix evaluation monitoring and processing AABB transformations, aquatic buoyant densities, and directional friction constants cleanly.
- **Lighting Subsystem:** radial gradient multiplication masking providing advanced ambient darkness overlays and lighting layers natively.
